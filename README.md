# FTP-CHAT
An Advanced Chat Program written in batch
![Image](https://i.postimg.cc/YSXXWRQ8/screenshot-62.png)


# Features
- Colored chat
- Instant script sharing
- Automatic updates
- File Sharing
# Coming soon
- Bug Reporting
- Help Files
- Online User List

# What you need
An FTP Server with no connection limit.
Nothing else!
